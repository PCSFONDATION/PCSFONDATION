Description de l'objet.

classe : SAFE

NOM : HDD-STIT-S-1

photo de la console du program HDD-STIT-S-1

HDD-STIT-S-1 est un program .exe (executable microsoft) qui peut être lancé sur toute les machines windows, 32bits et 64bits, il semble marcher sur l'invite de commande. Le
program ce connecterais à un réseau pour l'instant inconnue, des commandes peuvent y être effectué comme
stit-p-install-netuper qui nous connecterait à un reseau toujours inconnue mais avec une interface distincte désignant un autre program installé. La taille de ce program reste
extrèmement incoherent depart la taille du fichier qui est apparament impossible à afficher par toute les machines et système d'exploitaion windows utilisé.

Le program n'a tout de même aucun effet environemental ni sur le corps de la personne l'utilisant, les commandes testé non donné aucun succès elle sont alors rendu public dans le cadre de l'organisation.

Historique

HDD-STIT-S-1 à été trouvé dans un ordinateur
aujourd'hui classé SCP-713, sous forme d'un CD-ROM. Il à été placé dans un disque dur(HDD) de
l'organisation avec tout les autres program classé SAFE. il à aussi été modifié (impossible
de supprimer du code mais d'en rajouter) pour y ajouter '1.1.2.3 straight disposal' signe d'appartenance à l'organisation.
